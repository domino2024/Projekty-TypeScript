# Millionerzy Game

A quiz game similar to Who Wants to Be a Millionaire.

## Prerequisites

- Node.js 22 or higher
- npm 22 or higher

## Installation

1. Install npm 22 (if you haven't already)
2. Install dependencies:
   ```bash
   npm i
   ```

## Running the Application

Start the application using:
```bash
npm run start
```
